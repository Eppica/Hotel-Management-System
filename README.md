# Hotel Management System

## About
Hotel Management System is a program that implements functionalities, like:
- book 
- guest check-in
- guest check-out
- control payments 
- maintain guests 
- manintain rooms
- maintain room types
- mantain staff

It is a Web Application developed for the Programming Language class.

## Technologies

- Maven
- MySQL
- HTML
- CSS
- JSP
- JavaScript
- Maven
